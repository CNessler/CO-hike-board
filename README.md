# Overview

  Colorado Hike Board is an open-source application where a user is able to search for hikes, find out a bit about it, and also add their own hike.

# Technologies

  - Mongo
  - Express
  - Angular
  - NodeJS
  - Javascript
  
# Setup

  - express install
  - npm install
  - bower install angular(route, animate, and min.js)
